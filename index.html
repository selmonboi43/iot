Practical 01

Aim:

Starting Raspbian OS, Familiarizing with Raspberry Pi components and Interface, Connecting to Ethernet, Monitor, and USB

Introduction

Raspberry Pi is an economical computer that runs a Linux-based OS called Raspbian, which is optimized for Raspberry Pi devices. It allows users of all ages to explore computing and learn programming in languages like Python and Scratch.

Components of Raspberry Pi

  • The Raspberry Pi board
  • Power supply
  • Micro-SD card for storage and OS
  • Optional display, keyboard, mouse, and accessories

Interface of Raspberry Pi

The Raspberry Pi uses an HDMI connection for video output. Below are the primary interfaces available:

  • Raspbian Linux desktop
  • File explorer
  • Console
  • Browser
  • Raspberry Pi configuration tools

Connecting to Ethernet, Monitor, and USB

Follow these steps to connect your Raspberry Pi to Ethernet, monitor, and USB devices:

  • Insert your USB storage device into your computer, transfer necessary files, and plug it into the Raspberry Pi.
  • Connect an Ethernet cable to the Raspberry Pi's Ethernet port.
  • Connect a monitor using the HDMI port on the Raspberry Pi.

Practical 02

Aim:

Blinking LED using Raspberry Pi

Hardware Requirements:

  • Breadboard
  • LED
  • Resistor
  • Jumper Wires
  • Raspberry Pi
  • Power Supply

Steps

  1. Place the LED on the breadboard. The longer terminal is positive, the shorter is negative.
  2. Connect a jumper wire from the positive leg of the LED to a resistor.
  3. Connect another wire from the resistor to Raspberry Piâ€™s GPIO Pin 7 (positive) and GPIO Pin 9 (negative).

Code


import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BOARD)
GPIO.setup(7, GPIO.OUT)

for i in range(10):
    GPIO.output(7, True)
    print("LED is ON")
    time.sleep(1)
    GPIO.output(7, False)
    print("LED is OFF")
    time.sleep(1)

GPIO.cleanup()

Output

The LED will blink on and off 10 times.

Practical 03

Aim:

Displaying Different LED Patterns with Raspberry Pi

Hardware Requirements:

  • Raspberry Pi (Model B+)
  • 4 LEDs
  • 4 Resistors
  • Breadboard
  • BCM Connector
  • Jumper Wires

Steps

  1. Place the LEDs on the breadboard, ensuring the positive leg of each LED is connected to resistors.
  2. Connect jumper wires to GPIO pins 18, 23, 24, and 25 on the Raspberry Pi.

Code


import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)
GPIO.setup(18, GPIO.OUT)
GPIO.setup(23, GPIO.OUT)
GPIO.setup(24, GPIO.OUT)
GPIO.setup(25, GPIO.OUT)

try:
    while True:
        GPIO.output(18, GPIO.HIGH)
        GPIO.output(23, GPIO.HIGH)
        GPIO.output(24, GPIO.HIGH)
        GPIO.output(25, GPIO.HIGH)
        time.sleep(2)
        GPIO.output(18, GPIO.LOW)
        GPIO.output(23, GPIO.LOW)
        GPIO.output(24, GPIO.LOW)
        GPIO.output(25, GPIO.LOW)
        time.sleep(2)
except KeyboardInterrupt:
    GPIO.cleanup()

Output

The LEDs will blink in different patterns.

Practical 04

Aim:

Display Time Over 4-Digit 7-Segment Display Using Raspberry Pi

Hardware Requirements:

  • Raspberry Pi
  • 4-Digit 7-Segment Display Module
  • Breadboard
  • Jumper Wires
  • BCM Connector

Steps

  1. Take a Raspberry Pi Model B+.
  2. Connect the mouse and keyboard to the respective USB ports.
  3. Connect the Ethernet cable to the Ethernet connector port.
  4. Connect the HDMI cable to the HDMI port.
  5. Connect a USB cable to the power supply.
  6. Place the BCM properly in the center of the breadboard.
  7. Connect one end of the ribbon cable connector to the GPIO pins of the Raspberry Pi and the other end to the BCM mode on the breadboard.
  8. Connect the VCC pin of the 7-segment display to 5V, GND pin to ground, DIO to GPIO 24, and CLK to GPIO 23 on the Raspberry Pi.

Code


import sys
import time
import datetime
import RPi.GPIO as GPIO
import tm1637

Display = tm1637.TM1637(23, 24, tm1637.BRIGHT_TYPICAL)
Display.Clear()
Display.SetBrightness(1)

while True:
    now = datetime.datetime.now()
    hour = now.hour
    minute = now.minute
    second = now.second

    val = [(int(hour / 10)), (hour % 10), (int(minute / 10)), (minute % 10)]
    Display.Show(val)
    Display.ShowDoublepoint((second % 2))
    time.sleep(0.25)

Output

The current time will be displayed on the 4-digit 7-segment module.

Practical 05

Aim:

Controlling Raspberry Pi with Telegram

Hardware Requirements:

  • Raspberry Pi
  • Ethernet Cable
  • USB Cable for Power Supply
  • LED
  • Breadboard
  • Jumper Wires
  • BCM Connector
  • Ribbon Cable Connector

Steps

  1. Take a Raspberry Pi Model B+ and connect the mouse, keyboard, Ethernet, and HDMI cables.
  2. Set up the LED with a resistor and connect it to GPIO Pin 11 for the positive side and GND for the negative side on the Raspberry Pi.
  3. Install Telegram bot dependencies and set up a bot using BotFather on Telegram.

Code


import sys
import time
import telepot
import RPi.GPIO as GPIO

GPIO.setmode(GPIO.BOARD)
GPIO.setup(11, GPIO.OUT)

def on(pin):
    GPIO.output(pin, GPIO.HIGH)

def off(pin):
    GPIO.output(pin, GPIO.LOW)

def handle(msg):
    chat_id = msg['chat']['id']
    command = msg['text']
    if command == 'on':
        on(11)
    elif command == 'off':
        off(11)

bot = telepot.Bot('YOUR_BOT_TOKEN')
bot.message_loop(handle)

while True:
    time.sleep(10)

Output

The LED connected to the Raspberry Pi can be controlled via Telegram commands 'on' and 'off'.

Code

from time import sleep

camera.stop_recording()

Practical 07

Aim:

Interfacing Raspberry Pi with RFID

Hardware Requirements:

  • Raspberry Pi
  • RC522 RFID Reader
  • RFID Tags
  • Jumper Wires

Steps

  1. Connect the RFID sensor to the Raspberry Pi:
    • VCC to Pin 1 (3.3V)
    • GND to Pin 6
    • MOSI to Pin 19
    • MISO to Pin 21
    • RST to Pin 22
    • SCK to Pin 23
    • SDA to Pin 24
  2. Enable the SPI interface using sudo raspi-config.
  3. Install the necessary libraries for RFID: 
    sudo apt-get update && sudo apt-get upgrade
    sudo pip3 install mfrc522

Code - Writing on RFID Tag


import RPi.GPIO as GPIO
from mfrc522 import SimpleMFRC522

reader = SimpleMFRC522()

try:
    text = input("Enter new data: ")
    reader.write(text)
    print("Data written successfully!")
finally:
    GPIO.cleanup()

Code - Reading from RFID Tag


import RPi.GPIO as GPIO
from mfrc522 import SimpleMFRC522

reader = SimpleMFRC522()

try:
    id, text = reader.read()
    print(f"ID: {id}")
    print(f"Text: {text}")
finally:
    GPIO.cleanup()

Output

The RFID reader will read and write data to RFID tags.

Practical 08

Aim:

Raspberry Pi-Based Oscilloscope

Hardware Requirements:

  • Raspberry Pi
  • ADS1115 ADC Chip
  • Breadboard
  • Jumper Wires
  • Potentiometer

Steps

  1. Set up the ADS1115 ADC chip on the breadboard and connect it to the Raspberry Pi via SDA and SCL pins (GPIO 2 and GPIO 3).
  2. Connect the potentiometer to the ADS1115 on channel A0.
  3. Install the necessary libraries for the oscilloscope: 
    sudo apt-get install python3-pip
    pip3 install adafruit-ads1x15 matplotlib drawnow

Code


import time
import matplotlib.pyplot as plt
from drawnow import drawnow
import Adafruit_ADS1x15

adc = Adafruit_ADS1x15.ADS1115()
GAIN = 1
values = []

plt.ion()

def makeFig():
    plt.ylim(-5000, 5000)
    plt.title('Oscilloscope')
    plt.grid(True)
    plt.ylabel('ADC output')
    plt.plot(values, 'ro-', label='Channel 0')
    plt.legend(loc='lower right')

adc.start_adc(0, gain=GAIN)

try:
    while True:
        value = adc.get_last_result()
        values.append(value)
        drawnow(makeFig)
        time.sleep(0.5)
        if len(values) > 50:
            values.pop(0)
except KeyboardInterrupt:
    adc.stop_adc()

Output

The Raspberry Pi will display live potentiometer values as an oscilloscope graph.

Practical 10

Aim:

Fingerprint Sensor Interfacing with Raspberry Pi

Hardware Requirements:

  • Raspberry Pi
  • USB to Serial Converter
  • Fingerprint Module
  • Push Buttons
  • 16x2 LCD
  • 10k Potentiometer
  • Breadboard or PCB
  • Jumper Wires
  • LED (Optional)
  • Resistor (150 ohm - 1k ohm) for optional LED

Steps

  1. To install the fingerprint sensor library, run the following commands: 
    sudo apt-get update
    sudo apt-get install python-fingerprint
  2. Check the USB port to which the fingerprint sensor is connected using the command: 
    ls /dev/ttyUSB*
  3. Update the USB port number in the Python code accordingly.

Code


import time
from pyfingerprint.pyfingerprint import PyFingerprint
import RPi.GPIO as gpio

# Define GPIO pins
RS = 18
EN = 23
D4 = 24
D5 = 25
D6 = 8
D7 = 7
enrol = 5
delet = 6
inc = 13
dec = 19
led = 26
HIGH = 1
LOW = 0

gpio.setwarnings(False)
gpio.setmode(gpio.BCM)

# Setup GPIO pins
gpio.setup(RS, gpio.OUT)
gpio.setup(EN, gpio.OUT)
gpio.setup(D4, gpio.OUT)
gpio.setup(D5, gpio.OUT)
gpio.setup(D6, gpio.OUT)
gpio.setup(D7, gpio.OUT)
gpio.setup(enrol, gpio.IN, pull_up_down=gpio.PUD_UP)
gpio.setup(delet, gpio.IN, pull_up_down=gpio.PUD_UP)
gpio.setup(inc, gpio.IN, pull_up_down=gpio.PUD_UP)
gpio.setup(dec, gpio.IN, pull_up_down=gpio.PUD_UP)
gpio.setup(led, gpio.OUT)

try:
    f = PyFingerprint('/dev/ttyUSB0', 57600, 0xFFFFFFFF, 0x00000000)
    if not f.verifyPassword():
        raise ValueError('The given fingerprint sensor password is wrong!')
except Exception as e:
    print('Exception message: ' + str(e))
    exit(1)

def enrollFinger():
    print('Enrolling Finger')
    print('Waiting for finger...')
    while not f.readImage():
        pass
    f.convertImage(0x01)
    result = f.searchTemplate()
    if result[0] >= 0:
        print(f'Fingerprint already exists at position #{result[0]}')
        return
    print('Remove finger...')
    time.sleep(2)
    print('Waiting for same finger again...')
    while not f.readImage():
        pass
    f.convertImage(0x02)
    if not f.compareCharacteristics():
        print('Fingers do not match!')
        return
    f.createTemplate()
    position = f.storeTemplate()
    print(f'Fingerprint stored at position #{position}')

def searchFinger():
    print('Waiting for finger...')
    while not f.readImage():
        pass
    f.convertImage(0x01)
    result = f.searchTemplate()
    if result[0] == -1:
        print('No match found!')
    else:
        print(f'Found fingerprint at position #{result[0]}')

def deleteFinger(position):
    if f.deleteTemplate(position):
        print(f'Fingerprint at position #{position} deleted!')

# Main loop
while True:
    searchFinger()
    time.sleep(1)

Output

The Raspberry Pi will interface with the fingerprint sensor, allowing fingerprint enrollment, searching, and deletion functionalities.
