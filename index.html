Practical 01

Aim:

Starting RaspbianOS, Familiarizing with Raspberry Pi components and Interface, Connecting to Ethernet, Monitor, and USB

Introduction

Raspberry Pi is a very economical computer that runs a Linux Operating System. Raspberry Pi officially recommends the use of the Raspbian Operating System. It is a Debian-based OS, explicitly made for Raspberry Pi and hence named Raspbian.

Raspberry Pi is a low-cost, credit-card-sized computer that plugs into a computer monitor or TV and uses a standard keyboard and mouse. It is a capable little device that enables people of all ages to explore computing and to learn how to program in languages like Scratch and Python.

Components of Raspberry Pi

  • The Raspberry Pi board
  • Power supply
  • Micro-SD card for storage and OS
  • Optional display, keyboard, mouse, and accessories

Interface of Raspberry Pi

The Raspberry Pi uses an HDMI connection for video, so you can connect the monitor directly. Below are the interfaces of Raspberry Pi:

  • Raspbian Linux desktop
  • File explorer
  • Console
  • Browser
  • Raspberry Pi configuration tools

Raspberry Pi Communication Interfaces

Serial: The serial interface has Rx and Tx pins for communication with serial peripherals.

SPI (Serial Peripheral Interface)

  • MISO (Master In Slave Out)
  • MOSI (Master Out Slave In)
  • SCK (Serial Clock)
  • CE0 (Chip Enable 0)
  • CE1 (Chip Enable 1)

I2C (Inter-Integrated Circuit)

The I2C interface allows synchronous data transfer with two pins - SDA and SCL.

Connecting to Ethernet, Monitor, and USB

To connect to Ethernet, follow these steps:

  • Insert USB storage into your computer, transfer files, and move the USB to the Raspberry Pi.
  • Connect Raspberry Pi to the Ethernet port with an Ethernet cable.
  • Connect a monitor using the HDMI port of Raspberry Pi.

Practical 02

Aim:

Blinking LED using Raspberry Pi

Hardware Requirements:

  • Breadboard
  • LED
  • Resistor
  • Jumper Wires
  • Raspberry Pi
  • Keyboard
  • Mouse
  • HDMI Cable
  • Ethernet Cable
  • Power Supply
  • Male to Female Jumper Wire

Steps

  1. Connect the LED to the breadboard.
  2. LED has 2 terminals (the longer terminal is positive, and the shorter terminal is negative).
  3. Connect one side of the jumper wire below the LED positive side on the breadboard and the jumper wire below the LED longer terminal.
  4. Connect one end of the resistor on the breadboard right below the jumper wire connected to the LED positive side.
  5. The other end of the resistor should be connected anywhere else on the breadboard. Now, connect another jumper wire right above the second end of the resistor.
  6. Connect the positive side of the jumper wire to PIN 7 and the negative side to PIN 9 of the Raspberry Pi.

Code


import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BOARD)
GPIO.setup(7, GPIO.OUT)

for i in range(10):
    GPIO.output(7, True)
    print("LED IS FINALLY ON")
    time.sleep(1)
    GPIO.output(7, False)
    print("LED IS OFF")
    time.sleep(1)

print("PROGRAM COMPLETE")
GPIO.cleanup()

Output

The LED will blink on and off.

Practical 03

Aim:

Displaying Different LED Patterns with Raspberry Pi

Hardware Requirements:

  • Raspberry Pi (Model B+)
  • 4 LEDs
  • 4 Resistors
  • Breadboard
  • BCM
  • Jumper Wires
  • Ribbon Cable Connector

Steps

  1. Take a Raspberry Pi of Model B+.
  2. Connect the mouse and keyboard to their respective USB ports.
  3. Connect the Ethernet cable to the Ethernet connector port.
  4. Connect the HDMI cable to the HDMI port.
  5. Connect a USB cable to the power supply.
  6. Place the BCM in the center of the breadboard.
  7. Connect one end of the ribbon cable connector to GPIO pins on the Raspberry Pi and the other end to BCM mode on the breadboard.
  8. Place the jumper wire next to pin number 18, and similarly for pins 23, 24, and 25.
  9. Place resistors next to the attached jumper wires.
  10. Now place the LEDs on the breadboard, ensuring the positive leg of the LED is next to the resistor.
  11. Finally, connect the remaining jumper wire to the ground (GND) pin.

Code


import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)
GPIO.setup(18, GPIO.OUT)
GPIO.setup(23, GPIO.OUT)
GPIO.setup(24, GPIO.OUT)
GPIO.setup(25, GPIO.OUT)

try:
    while True:
        time.sleep(2)
        print("LED ON")
        GPIO.output(18, GPIO.HIGH)
        GPIO.output(23, GPIO.HIGH)
        GPIO.output(24, GPIO.HIGH)
        GPIO.output(25, GPIO.HIGH)
        time.sleep(2)
        print("LED OFF")
        GPIO.output(18, GPIO.LOW)
        GPIO.output(23, GPIO.LOW)
        GPIO.output(24, GPIO.LOW)
        GPIO.output(25, GPIO.LOW)
except KeyboardInterrupt:
    print("Program terminated by user")
finally:
    GPIO.cleanup()

Practical 04

Aim:

Display Time Over 4-Digit 7-Segment Display Using Raspberry Pi

Hardware Requirements:

  • Raspberry Pi
  • 4-Digit 7-Segment Display Module
  • Breadboard
  • Jumper Wires
  • BCM Connector

Steps

  1. Take a Raspberry Pi of Model B+.
  2. Connect the mouse and keyboard to their respective USB ports.
  3. Connect the Ethernet cable to the Ethernet connector port.
  4. Connect the HDMI cable to the HDMI port.
  5. Connect a USB cable to the power supply.
  6. Place the BCM in the center of the breadboard.
  7. Connect the ribbon cable connector to GPIO pins on the Raspberry Pi and BCM mode on the breadboard.
  8. Connect the VCC and GND pins of the display to the 5V and GND pins on the breadboard.
  9. Connect the DIO pin of the display to pin 24 and the CLK pin to pin 23 of the Raspberry Pi.
  10. Use the command gpio readall to verify pin connections.

Code


import sys
import time
import datetime
import RPi.GPIO as GPIO
import tm1637

Display = tm1637.TM1637(23, 24, tm1637.BRIGHT_TYPICAL)
Display.Clear()
Display.SetBrightness(1)

try:
    while True:
        now = datetime.datetime.now()
        hour = now.hour
        minute = now.minute
        second = now.second

        val = [(int(hour / 10)), (hour % 10), (int(minute / 10)), (minute % 10)]
        Display.Show(val)
        Display.ShowDoublepoint((second % 2))
        time.sleep(0.25)
except KeyboardInterrupt:
    Display.Clear()
    GPIO.cleanup()

  7. Connect one end of the ribbon cable connector to GPIO pins on the Raspberry Pi and the other end to BCM mode.
  8. Place a jumper wire on pin 11, and another jumper wire on the GND pin.
  9. Insert the LED into the breadboard.

Setting Up Telegram Bot

  1. Open the Telegram app on your mobile.
  2. Type "BotFather" and start it.
  3. Click on /newbot.
  4. Provide a name for your bot.
  5. You will receive a token after creating the bot.
  6. Follow the steps below to install the necessary libraries for Telegram bot on your Raspberry Pi:

sudo apt-get install python-pip3
sudo pip3 install telepot

Clone the Telegram bot repository:


git clone https://github.com/salmanfarisvp/TelegramBot.git

Code


import sys
import time
import random
import datetime
import telepot
import RPi.GPIO as GPIO

# LED control functions
def on(pin):
    GPIO.output(pin, GPIO.HIGH)
def off(pin):
    GPIO.output(pin, GPIO.LOW)

# Setup
GPIO.setmode(GPIO.BOARD)
GPIO.setup(11, GPIO.OUT)

# Telegram bot handler
def handle(msg):
    chat_id = msg['chat']['id']
    command = msg['text']
    print(f'Got command: {command}')
    if command == 'on':
        bot.sendMessage(chat_id, 'Turning LED ON')
        on(11)
    elif command == 'off':
        bot.sendMessage(chat_id, 'Turning LED OFF')
        off(11)

# Bot setup
bot = telepot.Bot('YOUR_BOT_TOKEN')
bot.message_loop(handle)
print('I am listening...')

while True:
    time.sleep(10)

Conclusion

We successfully controlled the Raspberry Pi with a Telegram bot to turn the LED on and off.

Interfacing Raspberry Pi with RFID

Hardware Requirements:

  • Raspberry Pi (Model B+)
  • RC522 RFID Reader
  • RFID Tags
  • Jumper Wires (Female to Female)
  • USB Cable for Power Supply

Steps

  1. Take a Raspberry Pi of Model B+.
  2. Connect the RFID sensor pins as follows:
    • VCC to Pin 1 (3.3V)
    • GND to Pin 6
    • MOSI to Pin 19
    • MISO to Pin 21
    • RST to Pin 22
    • SCK to Pin 23
    • SDA to Pin 24
  3. Enable SPI interface using sudo raspi-config.
  4. Reboot the Raspberry Pi using sudo reboot.

Code - Writing on RFID Tag


import RPi.GPIO as GPIO
from mfrc522 import SimpleMFRC522

reader = SimpleMFRC522()

try:
    text = input("Enter new data: ")
    print("Place your tag to write")
    reader.write(text)
    print("Data written successfully!")
finally:
    GPIO.cleanup()

Code - Reading from RFID Tag


import RPi.GPIO as GPIO

reader = SimpleMFRC522()

try:

finally:
    GPIO.cleanup()

Conclusion

Practical 08

Aim:

Raspberry Pi Based Oscilloscope

Hardware Requirements:

  • Raspberry Pi
  • ADS1115 ADC Chip
  • Jumper Wires
  • Breadboard
  • BCM
  • Potentiometer
  • USB Cable for Power Supply

Steps

  1. Take a Raspberry Pi of Model B+.
  2. Connect the ADS1115 chip to the breadboard.
  3. Connect jumper wires from the ADS1115 to Raspberry Pi GPIO as follows:
    • VDD to 3.3V
    • GND to GND
    • SDA to GPIO 2 (Pin 3)
    • SCL to GPIO 3 (Pin 5)
  4. Connect the potentiometer to the ADS1115 A0 channel.
  5. Install required libraries:

sudo apt-get install python3-pip
pip3 install adafruit-ads1x15
pip3 install matplotlib
pip3 install drawnow

Code


import time
import matplotlib.pyplot as plt
from drawnow import drawnow
import Adafruit_ADS1x15

adc = Adafruit_ADS1x15.ADS1115()
GAIN = 1
values = []

plt.ion()

def makeFig():
    plt.ylim(-5000, 5000)
    plt.title('Oscilloscope')
    plt.grid(True)
    plt.ylabel('ADC output')
    plt.plot(values, 'ro-', label='Channel 0')
    plt.legend(loc='lower right')

adc.start_adc(0, gain=GAIN)

try:
    while True:
        value = adc.get_last_result()
        print(f"Channel 0: {value}")
        values.append(value)
        drawnow(makeFig)
        plt.pause(0.001)
        time.sleep(0.5)
        if len(values) > 50:
            values.pop(0)
except KeyboardInterrupt:
    pass
finally:
    adc.stop_adc()
    plt.close()

Conclusion

We successfully created a Raspberry Pi-based oscilloscope to visualize ADC values from the potentiometer.

Practical 10

Aim:

Fingerprint Sensor Interfacing with Raspberry Pi

Hardware Requirements:

  • Raspberry Pi
  • Fingerprint Sensor
  • USB to Serial Converter
  • Jumper Wires
  • USB Cable for Power Supply

Steps

  1. Connect the fingerprint sensor to the Raspberry Pi using the USB to Serial converter.
  2. Install the fingerprint sensor library:

sudo apt-get install python-fingerprint

Code


import time
from pyfingerprint.pyfingerprint import PyFingerprint
import RPi.GPIO as GPIO

GPIO.setwarnings(False)
GPIO.setmode(GPIO.BCM)

try:
    f = PyFingerprint('/dev/ttyUSB0', 57600, 0xFFFFFFFF, 0x00000000)
    if not f.verifyPassword():
        raise ValueError('The given fingerprint sensor password is wrong!')
except Exception as e:
    print(f'Exception message: {e}')
    exit(1)

# Enroll fingerprint
def enroll_finger():
    print('Waiting for finger...')
    while not f.readImage():
        pass
    f.convertImage(0x01)
    f.createTemplate()
    positionNumber = f.storeTemplate()
    print(f'Finger enrolled successfully at position {positionNumber}')
    
# Search fingerprint
def search_finger():
    print('Waiting for finger...')
    while not f.readImage():
        pass
    f.convertImage(0x01)
    result = f.searchTemplate()
    positionNumber = result[0]
    if positionNumber == -1:
        print('No match found!')
    else:
        print(f'Found at position {positionNumber}')

# Main loop
while True:
    user_input = input("Enter 'e' to enroll, 's' to search, or 'q' to quit: ")
    if user_input == 'e':
        enroll_finger()
    elif user_input == 's':
        search_finger()
    elif user_input == 'q':
        break

Conclusion

We successfully interfaced the Raspberry Pi with a fingerprint sensor to enroll and search fingerprints.
